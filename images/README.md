# Dress Images

Add your dress product images in this folder.

Recommended names:

- floral-wrap-midi-dress.jpg
- satin-evening-gown.jpg
- cotton-a-line-dress.jpg
- black-bodycon-dress.jpg

You can use these image paths in `assets/js/dresses.js`, for example:

```js
image: 'images/floral-wrap-midi-dress.jpg'
```
